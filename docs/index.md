MinecraftForge文档
==================

这里是MinecraftForge的官方文档的中文翻译。[MinecraftForge]为Minecraft的Modding API。

这个文档 _仅仅_ 是Forge的教程，**而不是Java教程**。如果你不会Java，请先学习之后再来看这个文档。

可以在[GitHub]上投稿.
